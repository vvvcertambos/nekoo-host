UPLOADING FILES

Drag and drop files onto the homepage or click to select. Maximum file size: 2GB.

API USAGE

Upload any file via our simple API:

curl -X POST https://nekoo.ru/api/upload \
  -F "file=@yourfile.png"

Response (instant):

{
  "status": "success",
  "url": "https://cdn.nekoo.eu.org/12345678901234567890.png",
  "slug": "12345678901234567890.png",
  "id": 123,
  "hash": "abc123...",
  "size": 12345,
  "original_filename": "yourfile.png",
  "created_at": "2026-02-05T00:00:00Z"
}

FAST UPLOAD TECHNOLOGY

Our API returns your URL instantly (~50ms). The file is stored in our database immediately, and backup upload happens in the background. No waiting!

DEDUPLICATION

If you upload the same file twice, you get the same URL instantly. No duplicate storage.

FILE EXPIRATION (TTL)

Set a time-to-live via query parameter:

POST /api/upload?ttl=1h   (expires in 1 hour)
POST /api/upload?ttl=12h  (expires in 12 hours)
POST /api/upload?ttl=24h  (expires in 24 hours)
POST /api/upload?ttl=7d   (expires in 7 days)

SHAREX CONFIGURATION

Configure ShareX with:

  • Request URL: https://nekoo.ru/api/upload
  • Request Method: POST
  • File Form Name: file
  • Response URL: $json:url$
